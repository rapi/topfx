# OHLC Providers

Php version on laravel for building and fetching graphics

## Installation

## Usage

```bash
npm i
composer install
echo ''>database/ochl.sqlite
php artisan migrate
cp .env.example .env
php artisan key:generate
mkdir public/img/big/symbols -p
```

## Development

```bash
npm start
```

## What we use
* React
* Laravel
* D3.js for graphics
